Aplikasi Web Visit Magang Agronomi (Offline 100%)

Cara pakai:
1. Ekstrak file ZIP.
2. Buka index.html di browser modern (Chrome / Edge / Firefox).
3. Data peserta bawaan dari file terlampir sudah disertakan.
4. Semua data tersimpan lokal di browser menggunakan IndexedDB.
5. Untuk reset total data aplikasi, klik tombol "Reset semua data aplikasi".

Fitur utama:
- 100% offline.
- Master peserta dapat diunggah dari file .xlsx.
- Autosuggest peserta berdasarkan nama / NIK / unit.
- Satu laporan visit bisa memuat lebih dari 1 mentee.
- Laporan bisa diedit dan disimpan overwrite.
- Export laporan ke Excel .xlsx.
- Preview & kirim ringkasan visit ke WhatsApp.
- Light mode default + switch dark mode.
